# njcom/framework

Framework for building web-sites, web-applications, web-IDEs, command-line applications.

## Current status

Not ready (WIP). API is not stable.

* [![.github/workflows/test.yml](https://github.com/morpho-os/framework/actions/workflows/test.yml/badge.svg)](https://github.com/morpho-os/framework/actions/workflows/test.yml)

## Used technologies

### For development

1. Containerization: todo
1. Server-side/Backend:
    1. Nginx
    1. PHP
    1. MySQL/MariaDB: optional, can be used as DBMS
1. Client-side/Frontend:
    1. TypeScript: required to compile to JavaScript
    1. SCSS: required to compile to CSS
    1. HTML
1. Infra:
    1. Git
    1. PHP
    1. Node.js: required for TypeScript
    1. GNU Bash
    1. GNU Make

### For production

1. Server-side/Backend:
    1. Nginx
    1. PHP
    1. MySQL/MariaDB: optional
1. Client-side/Frontend:
    1. JavaScript
    1. CSS
    1. HTML

## Supported OSs

Only Linux, other OSs are not supported.
